# QuizGame

**An Interactive Multiple-Choice Quiz Application in Python**

A well-structured command-line quiz game developed as an intermediate-level Python project. The application demonstrates clean code architecture, proper function design, data management using dictionaries and lists, and robust user input handling.

## Features

- **Three Comprehensive Categories**:
  - General Knowledge (12 questions)
  - Science (12 questions)
  - Anime & Manga (12 questions)
- **Customizable Quiz Length** – Users can select the number of questions (1 to maximum available per category)
- **3-Strike System** – Game terminates early after three incorrect answers
- **Input Validation** – Handles invalid category selections and answer choices gracefully
- **Randomized Question Order** – Ensures no repeated questions within a session
- **Comprehensive Scoring** – Displays score as fraction, percentage, and letter grade with personalized feedback
- **User-Friendly Interface** – Clear instructions, formatted output, and smooth gameplay flow
